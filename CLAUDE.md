## ドメイン知識（正確なガイド内容を書くための背景）

OpenClaw は、Monolith Productions による 2D プラットフォーマー *Captain Claw* の
エンジンを C++ で再実装したものです。再実装されているのは **エンジンのみ** で、
プレイにはオリジナルゲームのアセット（`CLAW.REZ` アーカイブ）が必要です。
ハッキング/改造方法をドキュメント化する際は、以下の構成要素を把握しておくと役立ちます。

- **アセット** は `CLAW.REZ` 内にあり、オリジナル形式を使用します：`WWD`（レベル/ワールドデータ）、
  `PID`（画像）、`ANI`（アニメーション）、`PCX`（パレット）、`XMI`/MIDI（音楽）、`WAV`（効果音）。
- **ランタイムスタック** は一般的に C++ + SDL2（ウィンドウ/入力/オーディオ）、
  Box2D（物理）、設定用の軽量 XML パーサーで構成されます。
- **アーキテクチャ** は *Game Coding Complete* の系譜にある、アクター/コンポーネント型かつ
  イベント駆動の設計（Actor、各種コンポーネント、EventManager）です。

これらはあくまで方向づけであり、絶対視しないでください。形式やモジュール名はフォークによって
異なるため、ガイド内で事実として記述する前に **上流の OpenClaw ソースで仕様を確認してください**。
